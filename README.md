### install

1.打開 terminal，Clone 此專案至本機電腦

```
git clone https://github.com/tsouhare/todolist.git
```

2.開啟 terminal

```
cd todolist
```

3.安裝 npm 套件

```
npm install
```

4.開啟專案

```
npm run start
```

5.出現以下內容表示伺服器已啟動

```
Local:   http://localhost:3000/
```

6.前往以下位置使用現有的 Todo List 功能

```
Local:   http://localhost:3000/todo
```
